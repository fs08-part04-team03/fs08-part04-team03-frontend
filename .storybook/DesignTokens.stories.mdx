## 타이포그래피

### Font Family

- **Primary Font**: `Suit` (Sans-serif)
- **CSS 변수**: `--font-family-base`
- **Tailwind 클래스**: `font-sans`

### Font Sizes

| Size | CSS Variable | Tailwind Class | Usage |
|------|-------------|----------------|-------|
| 8px | `--text-8` | `text-8` | - |
| 10px | `--font-size-10` | `text-10` | - |
| 12px | `--font-size-12` | `text-12` | 작은 라벨, 캡션 |
| 13px | `--font-size-13` | `text-13` | - |
| 14px | `--font-size-14` | `text-14` | 본문 (작은) |
| 15px | `--text-15` | `text-15` | 본문 |
| 16px | `--font-size-16` | `text-16` | 본문 (기본) |
| 18px | `--font-size-18` | `text-18` | 부제목 |
| 20px | `--font-size-20` | `text-20` | 부제목 (큰) |
| 24px | `--font-size-24` | `text-24` | 제목 (작은) |
| 30px | `--font-size-30` | `text-30` | 제목 |
| 32px | `--font-size-32` | `text-32` | 제목 (큰) |
| 40px | `--text-40` | `text-40` | - |
| 44px | `--font-size-44` | `text-44` | 큰 제목 |
| 50px | `--font-size-50` | `text-50` | 큰 제목 |

### Font Weights

- **Normal**: `font-normal` (400)
- **Semibold**: `font-semibold` (600)

### Line Height & Tracking

- **Tracking**: `tracking--0.4` (자간 -0.4px)
- **Line Height**: 기본값 사용

## Spacing

기본 spacing 단위는 1px입니다.

**사용 가능한 Spacing:**
- `0px` ~ `50px` (1px 단위)
- `8px`, `12px`, `16px`, `20px`, `24px`, `32px`, `40px`, `48px`, `64px` 등

**CSS 변수:**
- `--spacing-0` ~ `--spacing-50`

**Tailwind 클래스:**
- `p-8`, `p-12`, `p-16`, `p-20` 등
- `m-8`, `m-12`, `m-16`, `m-20` 등
- `gap-8`, `gap-12`, `gap-16`, `gap-20` 등

## Z-Index Layers

| Layer | Value | CSS Variable | Usage |
|-------|-------|--------------|-------|
| Back | 0 | `--z-back` | 기본 레이어 |
| Base | 1 | `--z-base` | 기본 콘텐츠 |
| Dropdown | 10 | `--z-dropdown` | 드롭다운 메뉴 |
| Tooltip | 20 | `--z-tooltip` | 툴팁 |
| Modal | 30 | `--z-modal` | 모달 |
| Modal Dropdown | 35 | `--z-modaldropdown` | 모달 내 드롭다운 |
| Header | 40 | `--z-header` | 헤더/네비게이션 |
| Toast | 50 | `--z-toast` | 토스트 메시지 |
| Menu | 60 | `--z-menu` | 메뉴 |

## Shadows

| Shadow | CSS Variable | Usage |
|--------|--------------|-------|
| Toast | `--shadow-toast` | 토스트 메시지 |
| Dropdown | `--shadow-dropdown` | 드롭다운 메뉴 |
| Ambient | `--shadow-ambient` | 부드러운 그림자 |
| Depth | `--shadow-depth` | 깊이감 있는 그림자 |
| Card | `--shadow-card` | 카드 컴포넌트 |

## 반응형 Breakpoints

| Breakpoint | Width | CSS Variable | Tailwind Prefix |
|------------|-------|--------------|-----------------|
| Mobile | 0px | `--breakpoint-mobile` | 기본 (prefix 없음) |
| Tablet | 744px | `--breakpoint-tablet` | `tablet:` |
| Desktop | 1024px | `--breakpoint-desktop` | `desktop:` |

## 사용 예시

### 컬러 사용

```tsx
// CSS 변수 사용
<div style={{ backgroundColor: 'var(--color-gray-50)' }}>Gray 50</div>

// Tailwind 클래스 사용
<div className="bg-gray-50 text-gray-950">Gray 50 Background</div>
```

### 타이포그래피 사용

```tsx
// CSS 변수 사용
<p style={{ fontSize: 'var(--font-size-16)' }}>16px 텍스트</p>

// Tailwind 클래스 사용
<p className="text-16 font-sans">16px 텍스트</p>
<h1 className="text-32 font-semibold">큰 제목</h1>
```

### Spacing 사용

```tsx
// Tailwind 클래스 사용
<div className="p-16 m-20 gap-12">
  <div className="p-8">Padding 8px</div>
  <div className="p-16">Padding 16px</div>
</div>
```
